# Map Block

A block for embedding Google Forms with automatic height adjustment.

## Features
- Automatic height adjustment based on form content
- Responsive design
- Error handling for invalid URLs
- Clean, modern styling

## Usage
| Map |
|-----|
| https://docs.google.com/forms/d/e/1FAIpQLScvbRhBZde-W61cIUrr6O8SgfDY-70nNYiYS_sv6TCa2jxBQQ/viewform?usp=dialog |

## Configuration
- CSS Variables:
  - `--map-max-width`: Maximum width of the form container
  - `--map-border-radius`: Border radius of the form container
  - `--map-box-shadow`: Box shadow effect for the form

## Accessibility
- Iframe title attribute for screen readers
- Keyboard navigation support
- Responsive design for all screen sizes

## Performance
- Lightweight implementation
- Dynamic height adjustment
- No external dependencies

## Browser Compatibility
- Works in all modern browsers
- Fallback styling for older browsers

## Troubleshooting
- Ensure the Google Forms URL is correct and accessible
- Check console for any error messages
- Verify the form is publicly accessible
